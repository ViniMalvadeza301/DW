Guia de Recursos para Desenvolvimento Web

Para iniciar qualquer projeto Moderno, o [Node.js](https://nodejs.org/) é a base fundamental, permitindo a execução de JavaScript no servidor. O gerenciamento de dependências é feito através do [NPM](https://www.npmjs.com/) ou do [Yarn](https://yarnpkg.com/).

No desenvolvimento do Front-end, frameworks como [React](https://react.dev/), [Vue.js](https://vuejs.org/) e [Angular](https://angular.io/) dominam o mercado. Para a estilização, além do CSS puro, utilizamos o [Tailwind CSS](https://tailwindcss.com/) ou o [Sass](https://sass-lang.com/).

No ecossistema de testes e produtividade, o [Jest](https://jestjs.io/) é essencial para testes unitários, enquanto o [Postman](https://www.postman.com/) facilita o teste de APIs. Todo o código deve ser versionado no [GitHub](https://github.com/) e as consultas rápidas podem ser feitas no [MDN Web Docs](https://developer.mozilla.org/).
